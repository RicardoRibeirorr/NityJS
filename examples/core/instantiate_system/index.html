Instantiate System Example

This example demonstrates the Unity-like Instantiate system that automatically registers colliders and manages parent-child relationships.

Use WASD or Arrow keys to move the red player. Collisions with colored objects will be automatically detected!

Objects are created using Instantiate.create() which automatically:

  • Registers colliders with the CollisionSystem
  • Handles parent-child relationships
  • Adds objects to the current scene
  • Calls component start methods

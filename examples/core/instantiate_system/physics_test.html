Physics + Collision System Test

This demonstrates the separation between collision detection (CollisionSystem) and physics resolution (RigidbodyComponent).

Controls:

Use WASD or Arrow keys to move the red player.

Objects:

Red Player: Has RigidbodyComponent with 50% bounce. Will bounce off solid objects.
Blue Wall: Solid obstacle. Player will bounce off it.
Green Trigger Zone: Trigger collider only - no physics, just collision events.
Purple Moving Obstacle: Has gravity and 80% bounce. Can collide with other objects.

System Architecture:

CollisionSystem: Handles all collision detection and fires onCollisionEnter/Stay/Exit events.
RigidbodyComponent: Only handles physics resolution (bouncing, movement blocking) for non-trigger collisions.
Instantiate System: Automatically registers colliders and manages object creation.

Check the browser console for collision event logs!
